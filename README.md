# Crypto Time Series Analytics & Power BI Dashboard

This project demonstrates an end-to-end cryptocurrency analytics pipeline including data collection, preprocessing, time series forecasting, and interactive dashboard development using Power BI.

## 📌 Project Workflow
1. Real-time & historical Bitcoin data collection using Python  
2. Data preprocessing & transformation  
3. Time series forecasting using Facebook Prophet  
4. Interactive multi-page Power BI dashboard for business insights  

## 📂 Project Structure
- data → Historical & forecast datasets  
- scripts → Data collection & forecasting Python scripts  
- dashboard → Power BI interactive dashboard  
- documentation → Project screenshots & notes  

## 🛠 Tools & Technologies
- Python (Pandas, Prophet, yfinance)
- Power BI
- DAX
- Data Analytics
- Time Series Forecasting
